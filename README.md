# Receipt-detection-model-using-YOLOv8n-TFLite
receipt_model.tflite for SmartBill Android app.  Model: YOLOv8 Nano (Object Detection)  Format: TensorFlow Lite (Float32)  Resolution: 640x640  Training: 50 Epochs (~5 hours on Mac M4)  Detects: Merchant, Total, Date, Invoice details.
